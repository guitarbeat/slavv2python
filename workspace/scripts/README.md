# Scripts Directory

Internal scripts for system setup, CLI execution, and MATLAB integration.

## Directory Structure

- **[cli/](cli/)**: Core command-line tools and MATLAB wrappers.
  - `compare_matlab_python.py`: Main CLI tool for running comparison pipelines.
  - `run_matlab_cli.bat`: Windows batch wrapper for MATLAB.
  - `run_matlab_vectorization.m`: MATLAB entry point.
- **[setup/](setup/)**: Environment configuration and validation.
  - `setup_env.ps1`: Automated environment setup script.
  - `verify_imports.py`: Dependency verification tool.

## Interactive Analysis
For interactive work and tutorials, please use the **[notebooks/](../notebooks/)** directory in the repository root.
